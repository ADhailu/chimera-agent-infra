# Skill: Trend Fetch

## Purpose

Retrieve trending topics from social platforms.

## Input

{
"platform": "string",
"region": "string"
}

## Output

{
"topics": ["string"],
"confidence": 0.0
}

## Tools Used

- MCP Social API

# Description

This skill fetches trending topics from specified social media platforms using the MCP Social API. It takes in the platform and region as input parameters and returns a list of trending topics along with a confidence score. This skill is useful for staying updated on current trends and can be integrated into various applications that require real-time trend analysis.
